REST API는 상당수의 웹 서버 프레임워크에서 기본적으로 지원하는 기능으로 클라이언트가 웹 프로토콜을 기반으로 하여 효과적으로 데이터를 주고 받을 수 있도록 해준다. 이를 실습하기 위해 전체 고객 목록을 불러오는 API를 구현해 보자.

실제로는 DB에 연동해서 정보를 불러오는 형태로 개발해야 하지만 일단 하드코딩으로 해보자.
