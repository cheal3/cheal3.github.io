FAB

FAB (Floating Action Button)


사용 정의

화면상에 떠 있는 버튼으로, 서비스 전체에서 공통적으로 사용 될 수 있는 기능을 호출할 수 있다. 고정 상태의 버튼에 사용자의 액션(클릭, 터치)이 가해질 경우 애니메이션 효과가 가미된 지정된 이벤트가 발생한다.

사용 정의

CSS에서 position 속성값을 fixed로 설정하여 구현 가능 이후 디자인 적인 요소나 페이지 이동 이벤트 등을 연동시키는 방법으로 개발

크기

표준 버튼 : 56 x 56 dp

미니 버튼 : 40 x 40 dp

버튼 내 아이콘 크기 : 24 x 24 dp

DP : 디바이스 독립 픽셀
DPI가 커질수록 픽셀 값을 비례로 늘려 언제나 같은 크기로 보여질 수 있게함


고려할 점

긍정적인 액션을 위주로 직관적이고 구체적인 의미를 담은 아이콘을 사용하자!

Spin pic

물체가 회전하는 느낌을 주는 사진 기술, 파노라마가 촬영자를 중심으로 배경을 촬영하는 기법이라면 Spin pic은 피사체를 중심으로 촬영자가 주위를 돌며 촬영하는 기법 JS이미지 뷰어(라이브러리 有)를 사용, 드래그시 이미지를 넘겨 객체가 회전하는 느낌을 주는 방식으로 구현
